What is it?

Jeography GIS is a java program for using and manipulating OSM data/material. It does so by providing
  • An API with support for several different tasks like:
    • swing based maps and data visualization
    • geometric/geographic data processing
  • This GUI which is a viewer for maps and other GIS-like tasks:
    • show tiles from configurable sources via HTTP
    • make maps available offline
    • processing tiles to produce images or pdf-files of maps
    • viewing geometries of geographic entities
    • manipulating geometric data via GUI
    • address resolution
